# Development Documentation

This directory contains guides and documentation for developing with the Camp monorepo.

## Available Guides

### Setup & Configuration
- [ngrok Setup Guide](./ngrok-setup.md) - Configure ngrok for local development with external access
- [ChatGPT Dev Mode](./chatgpt-dev-mode.md) - Set up ChatGPT to work with your local MCP server

### Tools & Debugging
- [MCP Inspector](./mcp-inspector.md) - Use the MCP Inspector tool to test and debug your MCP server

### Testing & Quality
- [Verification Checklist](./verification-checklist.md) - Pre-deployment verification steps

## Getting Started

1. Start with the ngrok setup to enable external access to your local server
2. Configure ChatGPT Dev Mode to connect to your MCP server
3. Use the MCP Inspector to test your server implementation
4. Follow the verification checklist before deploying changes
